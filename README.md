# 基于区块链的众筹系统

这是一个参考 pet-shop Hardhat 实验结构完成的众筹 dApp。合约负责项目创建、捐赠、结束结算、成功提现、失败退款，并包含两个加分功能：达标后的阶段性资金释放、前 10 位早期捐赠者记录。

## 功能

- 创建项目：提交项目名称、描述、目标金额和截止日期，每个项目自动获得唯一 ID。
- 资金捐赠：用户选择进行中的项目捐赠 ETH，合约记录捐赠者地址和金额。
- 项目展示：前端展示进行中和已结束项目的名称、描述、目标金额、已筹金额、剩余时间、捐赠者和早鸟捐赠者。
- 截止结算：截止后任何人都可以调用结束函数。
- 成功提现：达到或超过目标后，项目发起人可以提现。
- 失败退款：未达到目标时，捐赠者可以取回自己的捐款。
- 加分功能：项目达到目标后，发起人可提前释放 30% 资金；前 10 位捐赠者会被记录为早鸟支持者。

## 运行

安装依赖：

```shell
npm install
```

运行测试：

```shell
npm test
```

启动本地区块链：

```shell
npm run node
```

新开终端部署合约到本地链：

```shell
npm run deploy:localhost
```

启动前端：

```shell
npm run dev
```

浏览器打开 `npm run dev` 输出的本地地址，使用 MetaMask 连接 `http://127.0.0.1:8545` 本地网络，并导入 Hardhat 控制台输出的测试账户私钥即可演示。
